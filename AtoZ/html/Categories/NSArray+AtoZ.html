az_each:

Iterates over all the objects in an array and calls the block on each object

- (void)az_each:(void ( ^ ) ( id obj , NSUInteger index , BOOL *stop ))block

Parameters

obj

(Block Parameter) this is the object in the array currently being enumerated over

index

(Block Parameter) this is the index of obj in the array

stop

(Block Parameter) set this to YES to stop enumeration, otherwise there is no need to use this

Discussion

Iterates over all the objects in an array and calls the block on each object

This iterates over all the objects in an array calling the block on each object until it reaches the end of the array or until the BOOL stop pointer is set to YES. This method was inspired by Ruby’s each method and works very similarly to it, while at the same time staying close to existing ObjC standards for block arguments which is why it passes a BOOL stop pointer allowing you to signal for enumeration to end.

Important! If block is nil then this method will throw an exception.

az_eachConcurrentlyWithBlock:

Enumerates over the receiving arrays objects concurrently in a synchronous method.

- (void)az_eachConcurrentlyWithBlock:(void ( ^ ) ( NSInteger index , id obj , BOOL *stop ))block

Parameters

index

(Block Parameter) the position of the object in the array

obj

(Block Parameter) the object being enumerated over

stop

(Block Parameter) if you need to stop the enumeration set this to YES otherwise do nothing

Discussion

Enumerates over the receiving arrays objects concurrently in a synchronous method.

Enumerates over all the objects in the receiving array concurrently. That is it will go over each object and execute a block passing that object in the array as a parameter in the block. This methods asynchronously executes a block for all objects in the array but waits for all blocks to finish executing before going on.
